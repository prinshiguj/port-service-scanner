# Advanced Port & Service Scanner with HTML Report

A professional Python-based **Port & Service Scanner** that scans open ports, identifies running services, performs banner grabbing, and generates a stylish HTML report automatically.

---

# 🚀 Features

- ✅ Multi-threaded Port Scanning
- ✅ Open Port Detection
- ✅ Service Detection
- ✅ Banner Grabbing
- ✅ Automatic HTML Report Generation
- ✅ Dark Theme HTML Report
- ✅ Fast Scanning
- ✅ Domain to IP Resolver
- ✅ Professional Terminal Output
- ✅ Reports Folder Auto Creation

## 1️⃣ Clone Repository

```bash

```

## 2️⃣ Open Project Folder

```bash
cd advanced-port-scanner
```

## 3️⃣ Run Scanner

